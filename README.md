# CareHeart: Supporting Dementia Caregivers

_A dashboard designed to support dementia patient caregivers in managing daily responsibilities and monitoring patient well-being._

Key features include:
* **AI Chat Window**: Allows caregivers to discuss the emotional challenges of 24/7 care and share updates about their patients. The AI then generates a report summarizing key points for doctors.
* **Community Connection**: A section that helps caregivers connect with others, share experiences, and receive AI-based recommendations for support and engagement.
* **Wellness Visualization**: Displays trends indicating whether the patientâ€™s condition is stable, improving, or declining.
* **Task Management**: A to-do list feature for tracking essential caregiving duties such as administering medications.
